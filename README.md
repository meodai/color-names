<h1 align = "center"> LiOS:Colors Data</h1>

LiOS:Colors Data is a fork of a collection of handpicked colors : [meodai/color-names](https://github.com/meodai/color-names).

## Features

It has everything upstream offers but have some specific improvements:-

- A `CSS` file with pre-assigned variables of `translucent colors` that can be used in frosted glass designs.
- Multiple `JSON` files categorized by shades for each color of `VIBGYOR` with `Black & White`.
- `dist/colors` contains multiple `fragments.json` file which are loaded via `manifest.json` instead of loading the whole `colornames.json`, this practice ensures :-
  
  - Low bandwidth usage
  - Low server cost
  - And low stress on client device

> Note: This repository shall only be used as a data source not as a tool.
